## koa2-server

It is a backend project about login with token and third-party usage.


## Init

Koa2 requires node v7.6.0 or higher for ES2015 and async function support.

Tips: You should open Mysql first.

```bash
npm run init
```

It will create database about user at locally. 

You can find the file about Mysql in `./server/db`.


## Getting started

```bash
npm run server
```
